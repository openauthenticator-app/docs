## Overview

Open Authenticator Docs is the public-facing website for the Open Authenticator project. It includes :

- the main landing page ;
- platform-specific download links ;
- FAQ ;
- contact page ;
- a translation page ;
- privacy policy and terms of service.

> [!TIP]
> If you like this project, consider starring it on GitHub !

## Requirements

Open Authenticator Docs is powered by [Nuxt](https://nuxt.com), therefore, you will need to have
[Node.js](https://nodejs.org) installed.

> [!NOTE]
> Open Authenticator Docs is designed to be statically generated and deployed to GitHub Pages.
> You can see the workflow in action [here](https://github.com/openauthenticator-app/docs/actions).

## Installation

To clone the repository and install dependencies, run in a shell :

```sh
git clone https://github.com/openauthenticator-app/docs.git
cd docs
npm install
```

## Local development

Start the development server :

```sh
npm run dev
```

The site will then be available at `http://localhost:3000`.

## Build and static Generation

Create a production build :

```sh
npm run build
```

Generate the static version of the site :

```sh
npm run generate
```

Preview the production build locally :

```sh
npm run preview
```
